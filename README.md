## R系列筆記

在GitHub中的，筆記的R Markdown原始檔。   
資料夾的編號和筆記的編號一致，可以去對應編號的資料夾，找尋自己想要的筆記原始檔(.Rmd)。   
例如：`R筆記 – (1)安裝R與RStudio`，所對應的原始檔路徑會是`R1/R1.Rmd`
